# Snakey Snake Game
## Introduction: 
    This gae is a cpp game made by Harry-1391453181620 Developer Group. 
    This programme will create a colorful gaming interface with no bound. 
    There will be a snake which will be controlled by the user.
    If the snake eats the food, it will grow longer and the score will increase.
    There will be other snakes controlled by the computer. If the snake collides with itself or other snakes, the game will be over.
## How to Play:
    1. Use the arrow keys to control the direction of the snake.
    2. Eat the food to grow longer and increase your score.
    3. Avoid colliding with yourself or other snakes.
    4. The game ends when a collision occurs.
    5. The final score will be displayed at the end of the game.
    6. The high score will be saved in a file named "highscore.txt".
